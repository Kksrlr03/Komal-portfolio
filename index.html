Quote Terminals ‚Äì Autonomous X (Twitter) Quote Bot

I built an autonomous system that posts motivational quotes on X (Twitter) every 5 minutes. The idea was simple: quotes are a great source of motivation, so what if AI could generate and share them automatically?

I used GPT-4 to generate new quotes and also pulled from sources like Wikipedia and my own SQLite database. Each quote was passed through a sentiment filter using HuggingFace's DistilRoBERTa to ensure it stayed positive and meaningful.

Once a quote was generated, it was checked against existing entries in the SQLite database. If it was unique, it would be posted and stored ‚Äî preventing any repeats in the future.

The final flow pushed the accepted quote into a DynamoDB queue, which was picked up by an IFTTT trigger that posted it on my X account called Quote Terminals.

The project ran successfully for weeks, tweeting fresh AI-generated or curated quotes at regular intervals. I later paused the project as IFTTT‚Äôs premium plan (~‚Çπ200/month) became unaffordable, but the automation worked flawlessly end-to-end.
